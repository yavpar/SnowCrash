# 🛡️ ft_snowcrash - Curso de Seguridad Informática desde Cero

Bienvenido. Este es un curso completo y progresivo sobre **ciberseguridad y explotación de binarios**, diseñado para estudiantes que comienzan desde nivel 0.

El proyecto **ft_snowcrash** de École 42 es la **puerta de entrada** a la serie de proyectos de seguridad. Aquí aprenderás los fundamentos que necesitas para proyectos más avanzados.

---

## 📚 Estructura del Curso

El curso está dividido en dos áreas principales:

### 1️⃣ **Teoría** (`01-teoria/`)
Conceptos fundamentales explicados paso a paso, con ejemplos concretos. **Se aprende la "por qué" detrás de todo.**

### 2️⃣ **Práctica** (`02-practicas/`)
Ejercicios progresivos que puedes realizar en tu máquina Fedora. Cada ejercicio refuerza lo aprendido en teoría.

---

## 🚀 Cómo usar este curso

### Opción A: Ruta Recomendada (Teoría + Práctica Integrada)
1. Lee la lección de teoría
2. Realiza los ejercicios asociados
3. Compara con la solución
4. Continúa con la siguiente lección

### Opción B: Aprender Toda la Teoría Primero
Si prefieres comprender todos los conceptos antes de practicar:
1. Lee todas las lecciones en `01-teoria/` en orden
2. Luego realiza todos los ejercicios en `02-practicas/`

## 🛠️ Requisitos Previos

### Sistema Operativo
- **Fedora** (última versión)
- Si estás en otra distribución, algunos comandos pueden variar

### Herramientas Necesarias
- Terminal/Shell
- `gcc` (compilador C)
- Herramientas de análisis: `gdb`, `objdump`, `strings`, `hexdump`
- *Más detalles en `SETUP.md`*

### Conocimientos Previos
**NINGUNO**. Este curso asume que:
- Sabes abrir una terminal
- Entiendes comandos básicos de Linux (si no, vemos esto primero)
- No sabes nada de seguridad, binarios, o memoria

Si necesitas refrescar Linux básico, véase `recursos/linux-basico.md`

## 🎯 Objetivos Generales del Curso

Al completar **ft_snowcrash**, entenderás:

1. ✅ Cómo funcionan los binarios en Linux
2. ✅ Cómo funciona la memoria de un proceso
3. ✅ Qué son las vulnerabilidades básicas
4. ✅ Cómo identificar y explotar estas vulnerabilidades
5. ✅ Cómo protegerse contra ellas
6. ✅ Cómo usar herramientas de análisis como `gdb` y `objdump`

Estos conocimientos son **la base** para todo lo que viene después en ciberseguridad.
